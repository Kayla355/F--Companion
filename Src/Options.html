The following tags can be used when creating a file or foldername(not case-sensitive):

[NAME]: The Name of the manga.

[SERIES]: The Series or parody the manga belongs to.

[ARTIST]: The Artist or Author that created the manga.

[LANGUAGE]: The Language that the manga is in.

[TRANSLATOR]: The Translator or Scanlator of the manga.

[TAGS]: The content tags of the manga.

[PAGE]: The current page of the manga (Can only be used in the filename)
